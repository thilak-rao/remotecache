# Security Policy

## Reporting a vulnerability

Report security issues privately through GitHub's
[Private Vulnerability Reporting](https://github.com/thilak-rao/nx-cache-server-bun/security/advisories/new).
Don't open a public issue for a suspected vulnerability.

Expect an acknowledgement within a few days. When a fix ships, a GitHub Security Advisory is published and the reporter gets credit.

## Supported versions

Only the latest published image and `main` get security fixes. Pin to a released tag (`ghcr.io/thilak-rao/nx-cache-server-bun:X.Y.Z`) and upgrade to pick up fixes.
