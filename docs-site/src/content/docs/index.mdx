In May 2026, Nx deprecated its four official self-hosted cache plugins, citing the CVE-2025-36852 (CREEP) cache-poisoning flaw and pointing teams toward Nx Cloud — a paid product. This server is the self-hosted endpoint alternative: a `readonly`/`full` token split that lets you architect around the cache-poisoning class without paying for managed infrastructure.
